在连接器接口参数配置中，参数的节点和属性的区别

![](https://myhelpdoc.oss-cn-heyuan.aliyuncs.com/mdimages/5df282aeb1de101a84b8274d7f8bbd3d.jpg)

**属性 ：**

{

"data": "{\"id\":1,\"name\":\"李华\"}"

}

**节点：**

{

"data": {

"id": 1,

"name": "李华"

}

}
