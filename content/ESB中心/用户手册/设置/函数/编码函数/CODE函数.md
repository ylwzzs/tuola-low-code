后台管理中心->ESB中心->动作流（或接口信息的成功失败）中，点击动态赋值框，左侧选择函数后，在右侧选择编码函数中的CODE函数，根据名称或第三方编码获取编码

说明：  
在指定编码类型中根据名称或第三方编码获取编码

语法：  
CODE(category, String)

参数：

category 编码类型 必填，支持选择

String 名称或第三方编码 必填
